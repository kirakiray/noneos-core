# NoneOS Core

### CN

NoneOS Core 作为下一代 NoneOS 的核心引擎进行开发，其主要目标如下：
- 精心打造代码量精简的底层引擎，筑牢系统运行根基。
- 用于搭建 ofajs 应用的开发环境与运行容器，实现高效开发与稳定运行。
- 后续还将具备兼容任意前端框架的能力，打造通用运行容器。
- 最终致力于构建一套极少更新甚至无需更新的核心代码，保障系统的长期稳定性。

### EN

NoneOS Core is developed as the core engine for the next-generation NoneOS, with the following main objectives:
- Carefully crafted to be a lightweight underlying engine, laying a solid foundation for system operation.
- Used to build development environments and runtime containers for ofajs applications, achieving efficient development and stable operation.
- Will subsequently feature compatibility with any frontend framework, creating a universal runtime container.
- Ultimately dedicated to building a core codebase that requires minimal or even no updates, ensuring long-term system stability.

### JP

NoneOS Coreは次世代のNoneOSのコアエンジンとして開発されており、主な目標は以下の通りです：
- コード量を精巧に削減した低レベルエンジンを慎重に構築し、システムの動作基盤を強固にします。
- ofajsアプリケーションの開発環境と実行コンテナの構築に使用され、効率的な開発と安定した動作を実現します。
- その後、任意のフロントエンドフレームワークとの互換性も備え、汎用的な実行コンテナの構築を目指します。
- 最終的には、極めて少ない更新頻度、あるいは更新が不要なコアコードを構築し、システムの長期的な安定性を確保することに努めています。
