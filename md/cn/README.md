# NoneOS Core

NoneOS Core 是一款轻量级核心引擎，专为运行 ofa.js 应用而设计。无需额外安装任何软件，即可一键启动、开发与调试 ofa.js 项目。

引擎源于 NoneOS3，经深度精简与架构重构，实现系统与应用的彻底解耦。

我们的愿景：打造一套通用、极简且跨平台的应用运行环境，使各类应用能够在不同操作系统上实现真正的无缝、高效与稳定运行。

## 核心特性

- **轻量化设计** - 精简的代码架构，移除冗余功能，保持核心引擎的精炼高效
- **高性能运行** - 优化的底层引擎，提供稳定高效的运行环境，确保应用流畅运行
- **完整兼容** - 全面支持 ofa.js 应用的开发与运行，无缝对接现有生态
- **高度扩展** - 模块化设计，具备兼容多种前端框架的能力，为未来扩展预留空间
